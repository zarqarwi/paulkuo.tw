## 原文摘要

Skillboss 創辦人曲曉音從矽谷一線觀察，歸納 Agent 創業四大方向：內容生成自動化、客戶開發自動化、AI 自我訓練基礎設施、一人公司主流化。最具指標性的信號：YC 取消「不投單人創辦人」規則，預測 2024-2025 年將出現首個一人十億美金獨角獸。人才評價標準重構——19 歲黑客松冠軍直接加入 xAI（憑作品而非學歷）。

## 關鍵概念

- **Agent 創業四大方向**：內容生成、客戶開發、AI 自我訓練、一人公司 → [[ai-agent-economy]]
- **一人公司（OPC）主流化**：YC 取消單人創辦人限制，Agent 工具鏈使單人創業風險顯著降低 → [[one-person-team]]
- **AI 自我訓練**：借鑑 AlphaGo Zero 模式，無需人類標註，AI 自我對弈訓練（待建頁面）
- **人才評價重構**：作品 > 學歷，強學習意願 + 強執行力 + AI 應用能力 → [[skill-development]]
