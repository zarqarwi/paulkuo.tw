## 原文摘要
Maltbot 創始人從職業倦怠復甦後轉向 AI 領域，經由 Cloud Code 契機創立社群，開發出跨平台即時通訊助手，具備多模態互動與本地模型支援。專案在 GitHub 上呈現歷史級增長，開啟「個人助理元年」，顛覆傳統應用程式生態。

## 關鍵概念
- **AI 個人助理**：透過即時通訊工具交互的智能助手，象徵應用程式層重構 → [[ai-agent-economy]]
- **模型無關性**：支援 OpenAI、Claude、本地開源模型的架構設計 → [[human-ai-collaboration]]
- **為模型而非人類構建**：設計理念強調符合 AI 思維模式的命令列優先策略 → [[build-for-models]]
- **提示注入（Prompt Injection）**：未完全解決的安全風險（待建）
