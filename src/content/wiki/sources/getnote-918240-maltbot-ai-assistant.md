## 原文摘要
Maltbot 創始人從職業倦怠復甦後轉向 AI 領域，經由 Cloud Code 契機創立社群，開發出跨平台即時通訊助手，具備多模態互動與本地模型支援。專案在 GitHub 上呈現歷史級增長，開啟「個人助理元年」，顛覆傳統應用程式生態。
