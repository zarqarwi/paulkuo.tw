# The Supervisor Class: AI Agents Are Remaking the Developer's Career

**By Fortune, March 2026**

Mohith Shrivastava (Principal Developer Advocate at Salesforce) argues that AI agents are fundamentally transforming software development. Developers are transitioning from writing code manually to supervising autonomous agents that handle implementation tasks.

## From Code Production to Orchestration

- Developers now focus on prompting systems with goals and requirements rather than typing every line of code
- The terminal becomes a central hub for overseeing autonomous workflows
- Value shifts from syntax memorization to high-level judgment

## Agent Skills as Persistent Knowledge

The industry is adopting modular, natural-language instructions called "agent skills" that solve the "forgetting problem"—where AI agents lose context. These create persistent project repositories of best practices and architectural rules.

## Quality Over Quantity Metrics

Organizations should measure "Agentic Work Units" (discrete tasks completed by AI) rather than lines of code or developer velocity, focusing instead on bug reduction and feature quality.

## Real-World Impact

- Lennar deploys 1.1 million agentic workflows monthly
- ReMarkable launched its first AI agent in three weeks, resolving 10,500+ inquiries
- Salesforce support agents autonomously handle 96% of cases

## Key Quotes

"The developer no longer just writes; they review, refine, and direct."

"Success should be measured by software quality: bug count reduction and architectural resilience."
