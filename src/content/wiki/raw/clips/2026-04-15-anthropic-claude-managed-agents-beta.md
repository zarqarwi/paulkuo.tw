On April 8, 2026, Anthropic launched a public beta service called **Claude Managed Agents** — a fully managed infrastructure to run autonomous Claude-based agents without anyone having to build or maintain the underlying technical layer.

This is Anthropic's equivalent of "serverless agents" — the orchestration, tool execution sandbox, persistence, and scaling are handled by Anthropic. Developers bring the agent definition (skills, system prompt, tools) and Anthropic runs it.

This launch sits alongside the Claude Agent SDK rename and signals that Anthropic is splitting its agent offering into two tiers: SDK for developers who want full control, Managed Agents for teams who want to ship without DevOps overhead.
