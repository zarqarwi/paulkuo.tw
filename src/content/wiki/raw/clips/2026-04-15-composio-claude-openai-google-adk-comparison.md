Composio compares the three dominant agent-building SDKs in 2026: Anthropic's Claude Agent SDK (renamed from Claude Code SDK), OpenAI's Agents SDK, and Google's ADK.

Claude Agent SDK's design principle is "give your agents a computer" — built-in tools for reading files, running bash commands, and editing code, so agents start working immediately without developers implementing tool execution. Programmable in Python and TypeScript.

The comparison covers tool registration ergonomics, context management, sub-agent orchestration, observability, and cost profiles across the three ecosystems.

**Wiki relevance**: Useful anchor for [[ai-skill-methodology]] and [[build-for-models]] — shows the agent SDK ecosystem has converged on a shared set of primitives (tools, context, sub-agents), which validates the "build for models" thesis that infrastructure is being shaped around agent consumption, not human consumption.
