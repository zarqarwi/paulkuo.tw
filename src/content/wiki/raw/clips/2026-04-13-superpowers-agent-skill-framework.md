Superpowers 是 GitHub 上由 obra 開發的開源框架，專為 coding agent 設計了一套完整的技能架構與軟體開發方法論。

核心特色：
- Skill-Based Architecture：透過可組合的「skills」定義 agent 能力
- 完整的端到端軟體開發生命週期工作流
- 模組化設計，允許將專業能力整合進 AI 驅動的開發任務
- 開源社群驅動開發

這是一套系統性的 agentic 軟體工程方法論，聚焦在可組合性和結構化工作流，為 AI coding agent 提供更高效的開發循環。
