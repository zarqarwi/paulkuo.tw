2026-04-08 發布的 Deep Agents v0.5 技術文章，介紹非同步子代理（AsyncSubagents）架構——主 Agent 可將任務委派給遠端背景程序並繼續執行其他平行邏輯，解決了傳統同步 Agent 工作流的效能瓶頸。關鍵技術特性包括：非阻塞子代理委派、多模態檔案系統支援、事件驅動狀態機制（Event-Driven State）取代輪詢。這反映了 Agentic 架構從簡單的 Chain-of-Thought 進化到複雜的多 Agent 協調系統。

**Wiki relevance**: 命中 [[human-ai-collaboration]] 和 [[ai-agent-economy]]——非同步多 Agent 架構是「人機協作」技術基礎的最新進展，展示 Agent 如何從單一對話走向多層級自主協調。
