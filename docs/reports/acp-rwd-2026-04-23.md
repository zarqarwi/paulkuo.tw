## 已知測試限制

### Target 03（`?id=SAMPLE_ID`）
`AICollabPortfolio` 是純 client-side React component（`client:load`）。headless puppeteer 的 `networkidle2` 在 JS hydration 尚未讀完 API 前即觸發，截圖只顯示初始表單狀態，**而非 results 頁（含雷達圖 + 分數條）**。這些截圖等同 target 01，無法用於 results view 驗證。

→ 建議：Paul 在手機實機上開 `https://paulkuo.tw/tools/ai-collab-portfolio/?id=7was4xtn` 做 results 肉眼確認。

### Target 04（`/portfolio/7was4xtn`）
此路由在 Astro 本機 dev 模式下返回 404，需要 Worker SSR 才能渲染。本次本機截圖無法驗證此目標。

→ 建議：Paul 在手機實機上開 `https://paulkuo.tw/portfolio/7was4xtn` 確認分享頁 RWD。
