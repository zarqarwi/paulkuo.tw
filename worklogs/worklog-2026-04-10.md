## 技術備忘
- Worker API 在 `api.paulkuo.tw`，不是 `paulkuo.tw`——所有前端呼叫 API 都用 `api.` subdomain
- 跨域 Bearer token 認證需要 CORS Allow-Headers 包含 `Authorization`
- governance-api.js 用 `new Response()` 時必須手動帶 `corsHeaders(request)`，或改用 `jsonResponse()`
- `governance-kv-seed.cjs` 在 GitHub Actions 裡要用 `npx wrangler`（不是裸 `wrangler`）
- Cowork sandbox 無法 curl 外部網站（proxy allowlist），線上驗證靠 Paul 或 Chrome MCP
- governance-api.js 用 Bearer token（env.GOVERNANCE_TOKEN）驗證，不用現有 auth/me 機制
- KV key 命名：gov:projects / gov:automation / gov:metrics:{project_id} / gov:summary
- Dashboard 純 Astro + canvas（無外部 chart lib），donut 用手繪 arc，bar chart 用 roundRect
